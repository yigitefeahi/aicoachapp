# Technology Role Specializations Knowledge Base

## Full Stack Developer
Full Stack Developer interviews evaluate frontend, backend, API design, database usage, deployment awareness, and the ability to connect product requirements across the stack. Strong answers mention end-to-end ownership, React or similar UI work, API contracts, authentication, data models, testing, and deployment tradeoffs.

## Machine Learning Engineer
Machine Learning Engineer interviews evaluate model development, feature engineering, model evaluation, data leakage prevention, deployment, monitoring, and production constraints. Strong answers explain metrics, baselines, offline versus online evaluation, model serving, drift, and experiment tracking.

## AI Engineer
AI Engineer interviews evaluate LLM applications, RAG, embeddings, prompt design, model selection, safety, cost/latency tradeoffs, evaluation, and integration with products. Strong answers explain retrieval quality, hallucination mitigation, guardrails, monitoring, and measurable user impact.

## Data Engineer
Data Engineer interviews evaluate ETL/ELT, data pipelines, orchestration, warehouses, streaming, data quality, lineage, partitioning, and reliability. Strong answers mention Airflow, Spark, Kafka, dbt, Snowflake, BigQuery, schema evolution, retries, and data validation.

## Cloud Engineer
Cloud Engineer interviews evaluate AWS/Azure/GCP services, networking, IAM, infrastructure as code, cost control, migration, reliability, and security. Strong answers discuss Terraform, VPC design, autoscaling, load balancing, monitoring, least privilege, and disaster recovery.

## Site Reliability Engineer
SRE interviews evaluate reliability, incident response, observability, SLO/SLA design, error budgets, automation, capacity planning, and postmortems. Strong answers include alert design, root cause analysis, on-call tradeoffs, rollback plans, and reducing toil.

## MLOps Engineer
MLOps Engineer interviews evaluate ML pipelines, model registry, CI/CD for models, feature stores, model monitoring, drift detection, reproducibility, and deployment automation. Strong answers connect ML quality with production reliability and governance.

## Database Administrator
Database Administrator interviews evaluate backups, replication, indexing, query tuning, access control, migrations, high availability, and recovery. Strong answers explain performance bottlenecks, monitoring, backup/restore testing, and data integrity.

## System Administrator
System Administrator interviews evaluate Linux/Windows administration, identity management, patching, scripting, backups, monitoring, and incident support. Strong answers include automation, security hygiene, documentation, and reliable operations.

## Solutions Architect
Solutions Architect interviews evaluate translating customer/business needs into technical architecture, integration design, cloud services, cost/risk tradeoffs, and stakeholder communication. Strong answers include constraints, alternatives, diagrams, and implementation roadmap.

## Software Architect
Software Architect interviews evaluate system design, modularity, technical leadership, domain boundaries, design patterns, scalability, maintainability, and architectural tradeoffs. Strong answers justify decisions and explain migration paths.

## Game Developer
Game Developer interviews evaluate game engines, gameplay systems, performance, physics, multiplayer, rendering basics, and debugging. Strong answers mention Unity or Unreal, frame rate, memory/performance constraints, player experience, and iteration.

## Embedded Software Engineer
Embedded Software Engineer interviews evaluate firmware, C/C++, microcontrollers, RTOS, hardware interfaces, memory constraints, timing, debugging, and reliability. Strong answers mention SPI/I2C/UART, interrupts, power constraints, and hardware-software validation.

## Blockchain Developer
Blockchain Developer interviews evaluate smart contracts, Solidity, Web3, security, gas optimization, DeFi concepts, wallets, and distributed systems tradeoffs. Strong answers discuss audits, reentrancy, testing, and transaction costs.

## Security Engineer
Security Engineer interviews evaluate application/cloud security, threat modeling, IAM, vulnerability management, secure coding, incident prevention, and security architecture. Strong answers explain risk severity, remediation, and prevention.

## Penetration Tester
Penetration Tester interviews evaluate vulnerability assessment, exploitation, OWASP, Burp Suite, Metasploit, Kali, reporting, and remediation guidance. Strong answers explain methodology, scope, evidence, impact, and responsible disclosure.

## SOC Analyst
SOC Analyst interviews evaluate SIEM, alert triage, log analysis, threat hunting, incident response, phishing/malware signals, and escalation. Strong answers mention Splunk or similar tools, investigation steps, and false positive reduction.

## Automation Engineer
Automation Engineer interviews evaluate scripting, workflow automation, RPA, test or process automation, integrations, reliability, and measurable time savings. Strong answers explain repetitive pain points, automation design, validation, and maintainability.

## Test Automation Engineer
Test Automation Engineer interviews evaluate automated test frameworks, Selenium, Cypress, Playwright, pytest/JUnit, CI integration, flaky test reduction, test data, and coverage strategy. Strong answers explain what to automate, why, and how to maintain tests.

## Scrum Master
Scrum Master interviews evaluate agile facilitation, sprint planning, retrospectives, impediment removal, team health, stakeholder alignment, and continuous improvement. Strong answers show servant leadership and measurable process improvements.

## Technical Project Manager
Technical Project Manager interviews evaluate planning, delivery, risk management, stakeholder communication, technical dependency tracking, and cross-functional execution. Strong answers explain scope control, tradeoffs, timelines, and escalation.

## Technical Support Engineer
Technical Support Engineer interviews evaluate troubleshooting, logs, customer communication, root cause analysis, escalation, documentation, and product feedback loops. Strong answers explain diagnosis steps and how issues were prevented later.

## IT Support Specialist
IT Support Specialist interviews evaluate helpdesk operations, hardware/software troubleshooting, Active Directory, ticketing, user support, device setup, and basic networking. Strong answers show clear communication and reliable resolution.

## Network Engineer
Network Engineer interviews evaluate TCP/IP, routing, switching, firewalls, VPNs, DNS, monitoring, troubleshooting, and network security. Strong answers explain packet flow, failure isolation, and configuration tradeoffs.

## ERP/CRM Consultant
ERP/CRM Consultant interviews evaluate business process mapping, requirements, configuration, implementation, user training, data migration, and stakeholder communication. Strong answers connect system configuration to business outcomes.

## SAP Consultant
SAP Consultant interviews evaluate SAP modules, implementation, configuration, business process fit, data migration, testing, user support, and change management. Strong answers mention relevant modules such as FICO, MM, SD, ABAP, or S/4HANA when supported by experience.
