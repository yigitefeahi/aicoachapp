# All Role Interview Profiles

This document summarizes role-specific interview themes and evaluation focus for the supported technology professions. It is used by RAG retrieval to ground questions, hints, CV feedback, and scoring.

Software Engineer: debugging, architecture tradeoffs, testing, maintainability. Evaluate problem solving, code quality, validation, and impact.
Full Stack Developer: end-to-end feature ownership, API contracts, frontend/backend tradeoffs, deployment. Evaluate full-stack reasoning, integration quality, product impact, and testing.
Backend Developer: API design, data modeling, reliability, performance, distributed systems. Evaluate system boundaries, database reasoning, observability, and failure handling.
Frontend Developer: UI architecture, state management, performance, accessibility, design collaboration. Evaluate user experience, component design, browser performance, and testing.
Mobile Developer: offline behavior, app performance, platform constraints, release quality. Evaluate mobile UX, device constraints, crash handling, and store release readiness.
Data Scientist: experimentation, model evaluation, statistical reasoning, business impact. Evaluate metric choice, data quality, model validity, and communication.
Machine Learning Engineer: feature engineering, model serving, model monitoring, ML pipeline reliability. Evaluate production ML, offline/online metrics, deployment tradeoffs, and drift handling.
AI Engineer: RAG quality, LLM evaluation, prompting, guardrails, latency and cost. Evaluate grounding, hallucination mitigation, AI product integration, and measurement.
Data Engineer: ETL pipelines, data quality, orchestration, warehousing, streaming. Evaluate pipeline reliability, schema evolution, data validation, and scalability.
Data Analyst: SQL analysis, dashboards, KPI definition, business insight, data cleaning. Evaluate metric clarity, stakeholder communication, insight quality, and recommendations.
Business Analyst: requirements gathering, process analysis, stakeholder alignment, UAT, acceptance criteria. Evaluate business clarity, documentation, prioritization, and validation.
Cloud Engineer: cloud architecture, IAM, networking, infrastructure as code, cost control. Evaluate security, reliability, cost awareness, and operational readiness.
Site Reliability Engineer: SLOs, incident response, observability, toil reduction, capacity planning. Evaluate reliability thinking, postmortem quality, automation, and risk management.
MLOps Engineer: model registry, CI/CD for ML, feature stores, model drift, reproducibility. Evaluate ML operations, monitoring, governance, and deployment safety.
Database Administrator: backup and recovery, query tuning, replication, indexing, access control. Evaluate data integrity, performance, availability, and restore confidence.
System Administrator: server operations, identity management, patching, monitoring, automation. Evaluate operational discipline, security hygiene, incident handling, and documentation.
Solutions Architect: customer requirements, solution design, integration, cost/risk tradeoffs. Evaluate architecture fit, stakeholder communication, constraints, and roadmapping.
Software Architect: domain boundaries, design patterns, technical direction, migration strategy. Evaluate architectural judgment, maintainability, leadership, and tradeoffs.
Game Developer: gameplay systems, engine performance, multiplayer, debugging, player experience. Evaluate performance, iteration, technical creativity, and user experience.
Embedded Software Engineer: firmware, RTOS, hardware interfaces, timing constraints, debugging. Evaluate resource constraints, reliability, hardware validation, and C/C++ reasoning.
Blockchain Developer: smart contracts, gas optimization, security audits, Web3 integration. Evaluate security, correctness, transaction cost, and testing.
Product Manager: roadmap tradeoffs, metrics, user research, prioritization, go-to-market. Evaluate product judgment, impact, stakeholder alignment, and measurement.
UX/UI Designer: user research, prototyping, interaction design, accessibility, design systems. Evaluate user empathy, design rationale, iteration, and communication.
DevOps Engineer: CI/CD, containers, infrastructure as code, release reliability, monitoring. Evaluate automation, deployment safety, observability, and collaboration.
Cybersecurity Analyst: threat analysis, SIEM, incident triage, vulnerability risk, security monitoring. Evaluate risk reasoning, investigation quality, prioritization, and communication.
Security Engineer: threat modeling, application security, cloud security, secure coding. Evaluate risk reduction, preventive controls, architecture, and remediation quality.
Penetration Tester: reconnaissance, exploitation, OWASP, reporting, remediation advice. Evaluate methodology, evidence, impact explanation, and ethics.
SOC Analyst: alert triage, log analysis, threat hunting, incident escalation. Evaluate investigation discipline, false positive reduction, response speed, and documentation.
QA Engineer: test strategy, regression risk, bug reporting, quality gates, manual and automated coverage. Evaluate risk-based testing, clarity, coverage, and collaboration.
Automation Engineer: workflow automation, scripting, RPA, integration, time savings. Evaluate process understanding, reliability, maintainability, and impact.
Test Automation Engineer: test frameworks, flaky test reduction, CI integration, test data, coverage. Evaluate automation strategy, maintainability, quality signal, and debugging.
Scrum Master: facilitation, team impediments, retrospectives, sprint planning, continuous improvement. Evaluate servant leadership, team health, process improvement, and conflict handling.
Technical Project Manager: delivery planning, dependency tracking, risk management, stakeholder communication. Evaluate execution, technical coordination, tradeoffs, and visibility.
Technical Support Engineer: troubleshooting, logs, root cause analysis, customer communication, escalation. Evaluate diagnosis, communication, resolution quality, and prevention.
IT Support Specialist: helpdesk, device setup, Active Directory, ticket handling, basic networking. Evaluate user empathy, troubleshooting, documentation, and reliability.
Network Engineer: routing, switching, firewalls, VPNs, packet flow, network troubleshooting. Evaluate network fundamentals, isolation strategy, security, and uptime.
ERP/CRM Consultant: requirements, configuration, business process, data migration, user training. Evaluate business fit, stakeholder management, implementation quality, and adoption.
SAP Consultant: SAP modules, configuration, implementation, data migration, process fit. Evaluate module knowledge, business process alignment, testing, and change management.
